== Installation ==

= Manual Installation =

1. Upload the entire `/fluxstore-api` directory to the `/wp-content/plugins/` directory.
1. Activate 'Fluxstore API' through the 'Plugins' menu in WordPress.

= Better Installation =

1. Go to Plugins > Add New in your WordPress admin and search for 'Fluxstore API'.
1. Click Install.
